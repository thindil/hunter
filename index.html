Hunter

Graphical Linux File Manager

Hunter is a graphical file manager for Linux, heavy inspired by Ranger. At this moment, Hunter is under heavy development and have only basic features of a file manager. To open files, the program uses default applications from XDG MIME Application Specification. The program is available only for Linux 64-bit.
